# web打印代理
## 简介
一款静默打印代理工具，通过建立Socket服务，接收打印指令，通过解析打印指令完成打印。
## 功能描述

### 系统功能
* 最小化到状态栏
* 单实例
* 保留异常日志
* 保留打印日志

### 打印功能
* 自定义纸张大小
* 手动分页
* 打印条形码，二维码
* 打印图片

## 通讯协议
使用JSON格式
web->代理
```
{
  id:1537148688,  // 唯一id，使用当前时间戳
  data:[],//打印指令
}
```
代理->web
```
{
  id:1537148688,
  status:0,//0成功，1失败
  msg:"返回的消息"
}
```
